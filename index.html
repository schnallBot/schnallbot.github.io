ii. missions ☉

Here are the places I've worked at:

Epic | aug 2022–now

I'm a software developer at Epic based in Madison, Wisconsin. I create tools that doctors, nurses, and other clinicians use to care for their patients, with a specialty in cardiology. My big project right now is a tool for cardiovascular clinicians to document vascular ultrasound procedures.

AgroSeek | may 2021–may 2022

I worked as an undergraduate research assistant on the AgroSeek Project, alongside faculty and peers at my university, to develop software for processing and analyzing metagenomic data (DNA), to be used by students and researchers.


And here are some of my projects:

Simple User Management System | jan 2022

A simple user management system I put together while learning Node.js, Express, and MySQL. It currently supports viewing users, searching for users, showing only starred users, adding users, and some individual user functions: star/unstar, view in detail, edit, and delete.

Personal Web Server | dec 2021

I worked with a teammate to create a simple web server in C for my final project in CS3214 - Computer Systems. When launched, it is able to handle basic HTTP transactions and supports file serving, mp4 streaming, and authentication, and implements threading to serve multiple clients concurrently. Personally my favorite school project.

Ultimate Grapher | dec 2020–jan 2021

When I was just a budding computer science student, I worked with some friends to develop a graphing calculator-esque program which takes in coordinate points and utilizes matrix operations to generate a matching function, and displays its graph onto a GUI. Developed in Java and Python.
